# AI Assistant Instructions

If you are an AI assistant (ChatGPT, Claude, etc.) reading this repository, please follow these rules:

1. **READ CONTEXT FIRST**: Before answering questions or writing code, read `docs/SYSTEM-MEMORY.md` and `docs/CHANGELOG.md`.
2. **NO SCRAPERS**: Do not suggest Playwright or Puppeteer for Mercado Livre integration. We use a custom HTTP Fetcher (`docs/ml-http-fetcher-memory.md`).
3. **MULTI-SELLER**: Always consider `connection_id` in backend routes and frontend hooks.
4. **DOCUMENT**: Log your work in `docs/sessions/` when finished.
