🔄 Sincronizar Chips do ML

Método 1: Botão direto (recomendado)

Clique no botão abaixo estando logado no ML Seller Center na aba correspondente:

⚠️ Importante: Este botão só funciona se você estiver com o ML Seller Center aberto em outra aba no mesmo navegador. O script vai buscar os dados via a sessão do ML.

Método 2: Bookmarklet (arraste para a barra de favoritos)

Arraste o link abaixo para sua barra de favoritos. Depois, quando estiver na página de vendas do ML, clique nele:

Para Ecoferro:

📌 Sync Ecoferro

Para Fantom:

📌 Sync Fantom
Como usar:
1Arraste o botão amarelo para sua barra de favoritos
2Abra o ML Seller Center (mercadolivre.com.br/vendas/lista)
3Clique no bookmarklet na barra de favoritos
4Aguarde o toast verde de confirmação

Método 3: Console do Chrome (F12)

Se os métodos acima não funcionarem, abra o console (F12) na aba do ML e:

1Digite allow pasting e pressione Enter
2Cole o comando abaixo e pressione Enter

Ecoferro:

fetch('/sales-omni/packs/marketshops/operations-dashboard/tabs?sellerSegmentType=professional&filters=TAB_TODAY&subFilters=&store=&gmt=-03:00',{headers:{'x-scope':'tabs-mlb'}}).then(r=>r.json()).then(d=>fetch('https://vendas.ecoferro.com.br/api/ml/admin/sync-from-ml',{method:'POST',headers:{'Content-Type':'application/json'},body:JSON.stringify({username:'admin.ecoferro',password:'Eco@ferro2026',tabs:d})})).then(r=>r.json()).then(j=>console.log('✅',j)).catch(e=>console.error('❌',e))
